Features Demonstrated

  • Konva.js Integration: Canvas-based image display with blocks and arrows
  • Drag & Drop: Words can be dragged from panel to blocks
  • Arrow Support: Visual arrows guide word placement
  • Real-time Validation: Immediate feedback on correct/incorrect placements
  • Progress Tracking: Visual progress bar and word count
  • Score Calculation: Automatic scoring with arrow compliance
  • Responsive Design: Works on different screen sizes
  • CSS Isolation: All styles scoped with .matching-test prefix
